1. Введение: 
    1) Наследование - принятие некого функционал или данные одного «класса» другим.
    2) Для того, чтобы создать класс, необходимо создать обычную функцию (в js любая функции которая вызывается с оператором new, являются конструктором).
    3) В js существует 2 вида наследования:
        1 - Прототипное
        2 - Функциональное

2. 
    1) Прототипный тип наследования:
        В js прототипное наследование реализуется через ссылку __proto__. При чтении свойства из обьекта, если его в нём нет, оно ищется в __proto__.
        Для того чтобы унаследовать класс в прототип дочернего класса, необходимо записать прототип другого класса.
        Object.create - создаёт обьект с указаным обьектом прототипа (создаётся обьект с ссылкой __proto__ в котором содержаться данные указанного параметра).

        После этого необходимо сохронить конструктор дочернего класса.(хз почему)

        Недостатки: не может получить доступ к своим приватным(локальным) данным, если только они не определены в контексте.

    2) Функциональныый тип наследования:
        Недостатки: функции в нём создаются заного при создании нового обьекта, сьедает больше памяти.
